# Marketspace

Projeto de Finalização de Módulo

## Descrição

Aplicativo de venda e compra no estilo OLX. Com ele, é possível cadastrar novos produtos a serem vendidos em seu perfil, além de poder ver os produtos à venda de outras pessoas.
